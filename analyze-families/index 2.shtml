Four Families

This page supports the Four Families activity from the Introducing Geometric Transformations as Functions unit. You will use Web Sketchpad to construct example functions from each of these families. You will observe each family to find similar behaviors, and you will compare the families with each other to find their differences.

This activity is a more open-ended, independent group-inquiry version of the individual family activities.

This activity is under construction! The worksheet and other support are not yet available.
